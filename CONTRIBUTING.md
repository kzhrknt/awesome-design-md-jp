# Contributing / コントリビューションガイド

日本語サービスの DESIGN.md 追加を歓迎します。

## DESIGN.md の追加手順

1. `template/DESIGN.md` をコピーして `design-md/<サービス名>/DESIGN.md` を作成
2. 実際のサイトのCSSを分析して正確な値を記述
3. 推測は避け、computed style や公開されているデザイントークンに基づく
4. Pull Request を作成

## 品質基準

### 必須

- **font-family のフォールバックチェーンが正確**であること（実サイトのCSSに基づく）
- **色は hex 値を正確に**記述すること
- **line-height, letter-spacing は実測値**を使うこと
- **セクションヘッダーは英語**（AIエージェントの可読性のため）
- **値の説明・Do's and Don'ts は日本語**で記述すること

### 推奨

- デザインシステムが公開されている場合はその公式ドキュメントを参照する
- 複数ページを確認して一貫した値を使う
- ライトモード・ダークモード両方をカバーする（対応している場合）

## ファイル構成

```
design-md/
└── <サービス名>/
    └── DESIGN.md
```

サービス名はローマ字小文字、ハイフン区切りとする（例: `smart-hr` ではなく `smarthr`）。

## セクション構成

`template/DESIGN.md` の9セクション構成に従ってください。特にセクション3（Typography Rules）のサブセクション 3.1〜3.8 は日本語サービスの核心部分です。

## コミットメッセージ

```
add: <サービス名> DESIGN.md
```
