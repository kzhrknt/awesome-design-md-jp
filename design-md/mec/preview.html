2026.04.25 — Press Release

「三菱地所と次にいこう。」新TVCM放映開始のお知らせ

人を、想う力。街を、想う力。三菱地所グループは、丸の内エリアを中心としたまちづくりで蓄積した知見を、次の時代の都市づくりへ広げていきます。

三菱地所株式会社（本社：東京都千代田区、執行役社長：中島篤）は、コーポレートメッセージ「人を、想う力。街を、想う力。」のもと、新たなTVCMを2026年4月25日（金）より全国で順次放映いたします。

本TVCMのねらい

本CMでは、三菱地所グループが取り組むまちづくりを、そこに集う人々の視点から描きました。建物そのものではなく、街を支える人とその想いに焦点を当て、コーポレートメッセージの核を伝えます。

放映期間: 2026年4月25日(金) 〜 2026年6月30日(月)
放映地域: 全国エリア
媒体: 民放各局・BS各局

三菱地所グループのまちづくり

三菱地所グループは、丸の内・大手町・有楽町エリアを中心に、約100年にわたるまちづくりを続けてきました。オフィス、商業、住宅、ホテル、空港、物流など、多様な領域で価値を提供しています。

本リリースに関するお問い合わせは、三菱地所株式会社 広報部までお願いいたします。

Font Note

フォントの注記

実サイトはグローバル領域で Century Gothic を先頭に置く独特な構成です。Century Gothic は macOS / Windows に標準搭載されているシステムフォントですが、Linux 系（puppeteer の headless Chrome を含む）には未搭載のことが多いため、このプレビューでは Jost（Google Fonts）を Century Gothic の代替として後段に置いています。

和文は Noto Sans JP を Google Fonts から weight 400 / 500 / 700 で読み込んでいます。実サイトでは下層ページで Noto Sans JP 先頭、ヘッダー・ナビ・トップ大型英字見出しで Century Gothic 先頭 と stack を切り替えています。

letter-spacing は全要素 normal、font-feature-settings も normal。和文の字詰め（palt）は使用せず、ベタ組で組まれています。和文見出しの行間は font-size × 1.4、本文は × 1.6〜2.0 が基本ルール。
